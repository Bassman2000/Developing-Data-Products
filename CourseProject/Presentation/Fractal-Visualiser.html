Introduction

The Shiny application allows one to view a selection of fractal images. The selection of fractals is not even close to being exhaustive.





Useful References

An introduction to fractals can be found in\(^{[1]}\). The code that calculates and plots the fractals here is modified from\(^{[2]}\).

  1. Fractal, Wikipedia, the free encyclopedia
  2. A Few Approaches to Generating Fractal Images in R, A. E. Voevudko, Code Project

Interface

Fractals of two types are identified by the way they are generated:

  1. Iterated Function Systems (IFS)
  2. Kronecker Products of Matrices (KP)

The user can select between IFS and KP types and then select a specific example of the chosen type from a drop down list.


Note:

The generation of some famous fractals, such as the Mandelbrot set and the Julia set, are significantly more resource intensive, and are not included here.

Example Simplified Algorithm

f = matrix(c(1,0,1, 
             0,1,0, 
             1,0,1), nrow = 3, ncol = 3)

mat <- f
for(i in 1:3) mat <- mat%x%f

m = nrow(mat) 
d = 0
X=NULL 
Y=NULL
  
for (i in 1:m) {
  for (j in 1:m) {
    if(mat[i,j]==0) next 
    else {
      d=d+1 
      X[d]=i; Y[d]=j 
    } 
  }
}
